# Estructura actual del proyecto

La versión organizada elimina archivos locales/generados y renombra assets para que sean más fáciles de ubicar.

- Templates de páginas: `backend/templates/pages/`
- CSS por página: `backend/static/css/`
- JavaScript del inicio: `backend/static/js/inicio.js`
- Imágenes organizadas: `backend/static/img/`
